## プライバシーポリシー / Privacy Policy

最終更新日 / Last Updated: 2026-04-29

### 収集する情報 / Data We Collect

本アプリ（ストレスリマインダー）は、以下のデータを**端末内にのみ**保存します：

- ストレス記録（日時・強度・カテゴリ・メモ）
- コーピング記録
- HealthKit / Apple Watch から取得したバイタルデータ（許可した場合）
- アプリ設定

これらのデータは外部サーバーに送信されることはありません。

This app (Stress Reminder) stores the following data **locally on your device only**:

- Stress logs (date, intensity, category, note)
- Coping entries
- Vitals data from HealthKit / Apple Watch (if authorized)
- App settings

None of this data is transmitted to external servers.

### 利用するサービス / Third-Party Services

- **Firebase Analytics**（Google LLC）: アプリ改善のため匿名の使用状況データを収集します。個人を特定する情報は含まれません。  
  For app improvement, anonymous usage data is collected. No personally identifiable information is included.  
  [Google Privacy Policy](https://policies.google.com/privacy)

- **Open-Meteo API**: 気象データ取得のため位置情報（緯度・経度）を送信します。  
  Location coordinates are sent to retrieve weather data.  
  [Open-Meteo Privacy Policy](https://open-meteo.com/en/terms)

### HealthKitについて / HealthKit

HealthKitのデータは、ストレス予報の精度向上のためのみ使用します。第三者と共有したり、広告目的で使用したりすることはありません。

HealthKit data is used solely to improve stress forecast accuracy. It is never shared with third parties or used for advertising.
